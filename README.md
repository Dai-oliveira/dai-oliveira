# Logo personalizado

Pequeno site com um logo estilizado (`Daiani Oliveira`). Ao clicar no nome, aparece uma mensagem de inspiração.

Como usar localmente:

1. Abra `logo-personalizado/index.html` no navegador (duplo clique) ou rode um servidor estático.

Para hospedar no GitHub:

1. Inicialize um repositório e faça commit:

```bash
git init
git add .
git commit -m "Add logo personalizado"
```

2. Crie um repositório no GitHub e faça push (substitua `origin`/`main` conforme necessário):

```bash
git remote add origin https://github.com/SEU_USUARIO/NOME_REPO.git
git branch -M main
git push -u origin main
```

Se quiser, posso criar o repositório remoto e fazer o push para você — me informe seu usuário GitHub e o nome do repositório, e confirme se posso criar o repositório público.
